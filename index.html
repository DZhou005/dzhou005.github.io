Projects

ClipIt

ClipIt is a user-friendly website where people can come together and upload their amazing gaming highlights via Amazon Web Services and share it with everyone on the website to see! While viewing the video, users can do various actions such as liking or commenting while on the video page. This was all done with Python in the backend and JavaScript in the frontend.

Technologies Used: React, Redux, Python, JavaScript, Flask, SQLAlchemy, CSS, AWS S3, Postgres

Queueflood

Inspired by Stackoverflow, Queueflood is a website for anyone to come to ask any JavaScript questions! Much like Stackoverflow, this website has an easy to use interface and users can navigate through the website with ease. There are various functions on the webpage such as searching through all the questions posted, answering questions, liking and disliking people's answers and much more. This website was created with JavaScript only. This website used Pug for the frontend and express for the backend.

Technologies Used: JavaScript, Express, Pug, Postgres, CSS, Sequelize, Postgres

QuickPics

QuickPics, much like Instagram, is a website for all users to upload images and share it with the user's followers. The Quickpics design is very similar to Instagram's design in terms of asthetics and functionality. This website has functions such as liking an image, commenting on a picture, searching for an user and much more. This was done with Python, Flask, and SQLALchemy in the backend and Javascript, React, and Redux in the front end.

Technologies Used: React, Redux, Python, JavaScript, Flask, SQLAlchemy, CSS, AWS S3, Postgres
